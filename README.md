一.介绍
1.项目描述
此项目为在线电商Web App的前台项目，包括首页, 搜索列表, 商品详情, 购物车, 订单, 支付, 用户登陆/注册等多个子模块
采用模块化、组件化、工程化的模式将静态页面和组件开发成vue模块。
